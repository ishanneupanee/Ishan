## Projects

### 1. Nepal VAT Liability Calculator
Calculates Output VAT, Input VAT, and determines whether
a business owes VAT to IRD or is eligible for a refund.
Covers goods, services, and mixed businesses at Nepal's
standard 13% VAT rate.

Built using: Python (variables, if/else, f-strings)
